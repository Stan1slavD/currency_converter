Тестовое задание

1. Переход между экранами осуществляется за счет свайпа влево или вправо
2. Обновление данных второго экрана происходит при переходе на него свайпом
3. "падение" или "подъем" валют по отношению к прошлому дню обозначено "+" и "-" соответственно
4. Базовая валюта на втором экране -USD
5. использовано открытое API https://currencylayer.com
